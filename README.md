# gnar-back-end
Backend using TypeORM NodeJS Express Multer Fast Csv and Postgress

# Database

## MAKE SURE YOU CREATE A POSTGRESS DATABASE WITH

### host: localhost
### port: 5432
### username: postgres
### password: postgres
### database: postgres

## Available Scripts

In order to install the dependencies run :

### `yarn install`

In the project directory, you can run:

### `yarn start`
